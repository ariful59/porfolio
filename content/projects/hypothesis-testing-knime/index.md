This project uses **KNIME** to build a visual workflow for hypothesis testing. It explores how **basic services**, **economic and social factors**, and **maternal mortality** relate to development status and life expectancy.

The project highlights practical statistical reasoning through a reproducible, low-code analytics workflow.
