This project integrates dermoscopic images with structured clinical information to improve skin disease classification. It uses pretrained CNN backbones and autoencoder-based feature extraction, then fuses visual and clinical representations through a neural classification pipeline.

Interpretability is an important part of the work, so the project includes Grad-CAM and SHAP to help explain which image regions and metadata features influence each prediction.
