This project packages an end-to-end ML workflow for structured healthcare data. It covers preprocessing, training, evaluation, and API deployment to support real-time patient risk assessment.

The goal is to keep the pipeline production-friendly while making the overall system easier to extend, test, and serve in downstream applications.
