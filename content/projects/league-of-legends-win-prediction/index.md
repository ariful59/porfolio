This project uses **machine learning and PyTorch** to predict League of Legends match outcomes from in-game statistics. It includes data preprocessing, exploratory data analysis, feature selection, and training of multiple classification models.

Model performance was evaluated with accuracy scores and confusion matrices, with results visualized using Matplotlib and Seaborn.
